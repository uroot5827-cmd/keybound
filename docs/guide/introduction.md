# 介绍

keybound 是一个**框架无关的键盘快捷键引擎**，从 TaskFlow 项目中提炼而来。

## 解决的问题

- 管理 100+ 快捷键的生命周期
- 解决快捷键在不同作用域的冲突
- 提供 Chord（两键序列）和多键序列支持
- 跨框架复用的基础设施

## 核心特性

| 特性       | 说明                                   |
| ---------- | -------------------------------------- |
| 注册表模式 | 单例注册表，动态增删改查               |
| 责任链     | 优先级调度，支持中断传播               |
| 作用域     | global / form / list / dropdown / edit |
| 序列键     | Chord + 3+ 键序列                      |
| 品牌类型   | ShortcutKey 编译时校验                 |
| 零依赖     | 核心引擎不依赖任何框架                 |
